# Ionic + Angular Project

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Build for production
npm run build:prod

# Run on Android
npm run android

# Run on iOS
npm run ios
```

## Project Structure

- `src/app/` - Angular root module
- `src/pages/` - Page components
- `src/services/` - Services
- `src/components/` - Reusable components
- `src/guards/` - Route guards
- `src/interceptors/` - HTTP interceptors
- `src/models/` - TypeScript interfaces
- `src/environments/` - Environment configs

## Tech Stack

- Ionic 8
- Angular 18
- Capacitor 6
- TypeScript 5.5
