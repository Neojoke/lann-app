## 📋 项目概览

### 项目名称
**Lann** (泰语：ล้าน - "百万")

### 项目描述
面向泰国用户的移动借款应用程序，提供便捷、安全、合规的个人贷款服务。

### 核心目标
1. 为泰国用户提供简单快速的在线借款体验
2. 支持泰语和英语双语界面
3. 符合泰国金融监管要求
4. 实现 Android 平台优先上线

## 👥 目标用户

### 主要用户群体
- **年龄:** 25-45 岁
- **地区:** 泰国主要城市（曼谷、清迈、普吉等）
- **收入:** 月收入 15,000-50,000 泰铢
- **特征:** 有短期资金需求，熟悉智能手机使用

### 用户痛点
1. 传统银行贷款流程繁琐、审批慢
2. 语言障碍（英文界面不友好）
3. 缺乏信用记录难以获得贷款
4. 担心高利贷和不透明收费

## 🎨 UI/UX 设计要求

### 设计风格
- **简约:** 大量留白，减少视觉干扰
- **高效:** 核心功能 3 步内完成
- **自然:** 友好亲和力，降低金融产品的距离感
- **泰式美学:** 融入泰国文化元素（颜色、图案）

### 颜色方案
- **主色调:** 金色 (#D4AF37) - 象征财富和信任
- **辅助色:** 深蓝色 (#1E3A8A) - 专业和安全
- **强调色:** 绿色 (#10B981) - 成功和通过
- **警告色:** 红色 (#EF4444) - 逾期和警告

### 字体选择
- **泰文字体:** Sarabun 或 Prompt (Google Fonts)
- **英文字体:** Inter 或 SF Pro
- **字号:** 适配泰语字符高度（比英文高 20%）

## 🏗️ 技术架构

### 前端技术栈
```
├── Ionic Framework 8.x (跨平台移动应用)
├── React 19.x (前端框架)
├── Capacitor 8.x (原生功能桥接)
├── Vite 5.x (构建工具)
└── TypeScript 5.x
```

### 后端技术栈 (Cloudflare)
```
├── Cloudflare Workers (无服务器函数 - API 层)
├── Cloudflare Pages (静态资源托管)
├── Cloudflare D1 (SQLite 数据库 - 用户数据)
├── Cloudflare R2 (对象存储 - 文档/图片)
├── Cloudflare Queue (异步任务队列)
└── Cloudflare Turnstile (人机验证)
```

### 第三方服务
```
├── SMS Gateway (泰国短信服务)
├── e-KYC 服务 (身份验证)
├── 人脸识别 API (活体检测)
├── 银行 API 接口 (放款/扣款)
├── Google Firebase (推送通知)
└── Sentry (错误监控)
```

## ⚠️ 风险管理

### 高风险
1. **合规风险:** 泰国金融牌照申请周期长
   - **缓解:** 提前咨询律师，准备备选方案（与持牌机构合作）
   - **状态:** 🟡 待确认

2. **技术风险:** Cloudflare D1 数据库性能瓶颈
   - **缓解:** 设计阶段进行压力测试，准备迁移方案
   - **状态:** ✅ 已识别

### 中风险
3. **第三方依赖:** 短信/KYC/银行 API 稳定性
   - **缓解:** 选择多家供应商，实现熔断降级
   - **状态:** ✅ 已规划

4. **语言本地化:** 泰语翻译准确性
   - **缓解:** 聘请泰国本地翻译，进行 A/B 测试
   - **状态:** ✅ 已配置

### 低风险
5. **用户体验:** 文化差异导致的设计问题
   - **缓解:** 邀请泰国用户参与可用性测试
   - **状态:** ✅ 已规划
