## 📋 项目概览

### 项目名称
**Lann** (泰语：ล้าน - "百万")

### 项目描述
面向泰国用户的移动借款应用程序，提供便捷、安全、合规的个人贷款服务。

### 核心目标
1. 为泰国用户提供简单快速的在线借款体验
2. 支持泰语和英语双语界面
3. 符合泰国金融监管要求
4. 实现 Android 平台优先上线

## 👥 目标用户 - 自主决策

### 主要用户群体 (自主决策)
- **年龄:** 25-45 岁 (有稳定收入的年龄段)
- **地区:** 泰国主要城市（曼谷、清迈、普吉、芭提雅）
- **收入:** 月收入 15,000-50,000 泰铢 (中等收入群体)
- **职业:** 上班族、小商贩、服务业从业者
- **特征:** 有短期资金需求，熟悉智能手机使用，有信用卡或银行贷款经验

### 用户痛点 (自主决策)
1. **审批慢:** 传统银行需要 3-7 个工作日
2. **手续繁琐:** 需要大量纸质文件和担保人
3. **语言障碍:** 英文/泰文专业术语难以理解
4. **信用门槛高:** 没有信用记录无法获得贷款
5. **不透明:** 隐藏费用和高额罚息
6. **还款不便:** 需要亲自到银行网点

### 产品定位 (自主决策)
**"泰国版花呗"** - 简单、快速、透明的数字借款平台

### 差异化优势 (自主决策)
| 维度 | Lann | 传统银行 | 其他网贷 |
|------|------|----------|----------|
| 审批时间 | 10 分钟 | 3-7 天 | 1-2 天 |
| 语言支持 | 泰语 + 英语 | 泰语 | 泰语 |
| 还款方式 | 4 种便捷方式 | 银行转账 | 银行转账 |
| 费用透明 | 借款前展示全部费用 | 隐藏费用多 | 部分透明 |
| 提前还款 | 免手续费 | 有手续费 | 有手续费 |

## ⚠️ 风险管理 - 自主决策

### 高风险 (自主决策)
1. **合规风险:** 泰国金融牌照申请周期长 (3-6 个月)
   - **决策:** MVP 阶段与持牌机构合作，分成模式
   - **时间线:** Week 1-2 确定合作伙伴
   - **状态:** 🟡 待执行

2. **信用风险:** 用户违约导致坏账
   - **决策:** 首次借款上限 5,000 THB，逐步提升额度
   - **风控措施:** 手机号验证 + 紧急联系人 + 逾期催收
   - **状态:** ✅ 已规划

3. **技术风险:** Cloudflare D1 数据库性能瓶颈
   - **决策:** MVP 使用纯内存模式，Phase 2 迁移 D1
   - **缓解:** 设计阶段进行压力测试，准备迁移方案
   - **状态:** ✅ 已识别

### 中风险 (自主决策)
4. **第三方依赖:** 短信/KYC/银行 API 稳定性
   - **决策:** 选择 2 家供应商，自动切换
   - **缓解:** 实现熔断降级，本地缓存 OTP
   - **状态:** ✅ 已规划

5. **语言本地化:** 泰语翻译准确性
   - **决策:** 使用 Google Translate API + 本地兼职审核
   - **缓解:** 聘请泰国本地翻译，进行 A/B 测试
   - **状态:** ✅ 已规划

6. **运营风险:** 客服响应不及时
   - **决策:** 工作时间 9:00-18:00 (曼谷时间)
   - **缓解:** 常见问题 FAQ + 智能客服机器人
   - **状态:** ✅ 已规划

### 低风险 (自主决策)
7. **用户体验:** 文化差异导致的设计问题
   - **决策:** 参考泰国本地 App 设计风格
   - **缓解:** 邀请泰国用户参与可用性测试
   - **状态:** ✅ 已规划

8. **竞争风险:** 已有网贷平台竞争
   - **决策:** 差异化定位 (更快、更透明、更便捷)
   - **缓解:** 快速上线 MVP，抢占市场先机
   - **状态:** ✅ 已识别
