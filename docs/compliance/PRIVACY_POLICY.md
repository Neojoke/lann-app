## 引言

Lann ("我们"、"我们的"或"本应用") 尊重您的隐私并致力于保护您的个人数据。本隐私政策将向您说明我们如何收集、使用、存储和保护您的个人数据，以及您在泰国《个人数据保护法》(PDPA) 下的权利。

请仔细阅读本隐私政策。使用 Lann 应用程序即表示您同意本隐私政策中描述的数据处理做法。

## 数据共享与披露

**Lann 不会向任何第三方出售您的个人数据。**

### 数据共享对象

| 共享对象 | 共享数据 | 共享目的 | 法律依据 |
|---------|---------|---------|---------|
| **持牌金融机构** | 身份信息、信用数据 | 资金合作、联合风控 | 合同履行 |
| **支付服务提供商** | 财务信息、交易数据 | 放款/还款处理 | 合同履行 |
| **短信服务商** | 手机号、OTP 数据 | 身份验证 | 合同履行 |
| **KYC 服务商** | 身份信息、生物识别数据 | 身份验证 | 法定义务 |
| **信用报告机构** | 信用数据、还款记录 | 信用报告 (NCIB) | 法定义务 |
| **监管机构** | 交易数据、身份信息 | 监管报告 (BOT/SEC) | 法定义务 |
| **技术服务商** | 设备信息、日志数据 | 系统运维、安全防护 | 合法利益 |

### 数据跨境传输

**原则:** 我们优先在泰国境内存储和处理数据。

**跨境场景:**
- Cloudflare 全球 CDN (可能涉及数据出境)
- 备份服务器 (可能位于新加坡/日本)

**保障措施:**
- 采用标准合同条款 (SCCs)
- 确保接收方具有同等数据保护水平

## 数据存储与保护

### 存储位置

| 数据类型 | 存储位置 | 存储方式 |
|---------|---------|---------|
| **用户数据** | 泰国 AWS 区域 / Cloudflare D1 | 加密存储 |
| **日志数据** | Cloudflare Logs | 加密存储 |
| **备份数据** | AWS S3 (泰国/新加坡) | 加密存储 |

### 安全措施

| 措施 | 描述 |
|------|------|
| **传输加密** | TLS 1.3 加密所有数据传输 |
| **存储加密** | AES-256 加密敏感数据 (身份证号、银行账号) |
| **访问控制** | 基于角色的访问控制 (RBAC) |
| **日志审计** | 记录所有数据访问操作 |
| **定期审计** | 每季度进行安全审计 |
| **员工培训** | 所有员工接受 PDPA 培训 |

### 数据泄露响应

**流程:**
1. **发现泄露** - 安全团队确认泄露事件
2. **评估风险** - 评估影响范围和严重程度
3. **通知监管** - 72 小时内向 PDPC 报告
4. **通知用户** - 高风险泄露及时通知受影响用户
5. **整改修复** - 修复漏洞并防止再次发生

## 您的权利

根据泰国 PDPA，您享有以下权利：

### 1. 访问权 (Right of Access)
- 请求访问我们持有的您的个人数据
- 获取数据副本

### 2. 更正权 (Right to Rectification)
- 请求更正不准确或不完整的个人数据

### 3. 删除权 (Right to Erasure)
- 在特定条件下请求删除个人数据
- 法律要求保留的数据除外

### 4. 限制处理权 (Right to Restrict Processing)
- 在特定条件下请求限制数据处理

### 5. 数据可携权 (Right to Data Portability)
- 请求以结构化、通用格式获取数据
- 请求将数据传输给其他控制者

### 6. 反对权 (Right to Object)
- 反对基于合法利益的数据处理
- 反对直接营销

### 7. 撤回同意权 (Right to Withdraw Consent)
- 随时撤回已给予的同意
- 不影响撤回前处理的合法性
