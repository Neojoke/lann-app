## Charter Lite v1.0 - 核心功能开发

**创建日期:** 2026-03-16 10:27  
**状态:** 🟡 开发中  
**优先级:** P0

### 项目目标
完成 Lann App MVP 核心功能开发：
1. 用户认证模块（登录/注册/OTP）
2. 借款流程（金额选择/期限选择/费用计算/电子合同）
3. 还款流程（还款计划/多种还款方式/提前还款）

### 技术约束
- **前端:** Ionic 8 + React 19 + Vite + TypeScript
- **后端:** Cloudflare Workers + Hono + 纯内存模式 (MVP)
- **数据库:** 内存存储 → Phase 2 迁移 D1
- **语言:** 泰语 + 英语双语

### 业务参数（已决策）
| 参数 | 值 |
|------|-----|
| 借款金额 | 1,000-50,000 THB |
| 借款期限 | 7/14/21/30 天 |
| 日利率 | 1.0% (年化 36.5%) |
| 逾期费率 | 日息 0.5% |
| 提前还款 | 免手续费 |
| 审批时效 | 10 分钟自动审批 |
| 放款时效 | 审批后 1 小时内 |
